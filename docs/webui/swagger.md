# Interactive API (Swagger)

This page embeds the Torrentarr WebUI API specification. You can browse endpoints and try requests (when **Try it out** is enabled, requests run against the URL you configure; they are not sent to any server by default).

The spec is generated from the running application. To refresh it after API changes, see [Regenerating the OpenAPI spec](../development/release-process.md#regenerating-the-openapi-spec).
